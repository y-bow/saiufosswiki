# Hands-on Git & GitHub Workshop for Beginners

**Date:** September 17, 2025  
**Format:** In-person  
**Venue:** SAI University, Chennai

## Overview

The FOSS Club of Sai University hosted an engaging hands-on 
workshop on Git and GitHub, bringing together students eager to 
strengthen their foundations in version control and collaborative 
development.

The session walked participants through core Git concepts, practical 
workflows, and real-world collaboration using GitHub. From 
understanding repositories and commits to exploring how teams 
contribute to shared codebases, the workshop emphasized why version 
control is indispensable in modern software development.

Students actively experimented, asked thoughtful questions, and 
gained confidence using tools that power both open-source and 
industry projects — reinforcing the values of collaboration, 
transparency, and shared ownership central to open source.
