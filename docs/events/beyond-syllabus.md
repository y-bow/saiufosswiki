# Beyond the Syllabus: Open Source in the Real World

**Date:** January 30, 2026
**Format:** In-person Workshop
**Speaker:** Siddharth Shivkumar
**Organized by:** FOSS Club, Sai University

## Overview

A practical session on how open source actually works beyond the classroom, open to students from all departments.

## What We Covered

- Git & GitHub fundamentals
- How pull requests and contributions work in real projects
- Using MkDocs for documentation
- How these tools are used in real-world open-source projects
- Discussion inspired by the WikiSyllabus / BeyondSyllabus initiative

## Key Takeaway

Open source is not just a development methodology — it's a learning culture. This session was a strong step towards building that culture at Sai University.
