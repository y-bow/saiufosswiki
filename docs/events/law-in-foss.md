# Law in FOSS: Who Owns Open Code?

**Date:** November 23, 2025
**Format:** Online Session
**Organized by:** FOSS Club, Sai University

## Overview

A thought-provoking online session exploring how legal frameworks intersect with collaborative software development.

## What We Covered

- Copyright in software and what "ownership" means in open source
- Why patenting code is often impractical
- Case study: LinkedIn vs HiQ — connecting global legal debates to the Indian context
- Legal implications of sharing code on platforms like GitHub
- How law can both protect and shape open collaboration

## Key Takeaway

Developers need legal awareness. Publishing code publicly isn't legally neutral — understanding licenses, copyright, and platform terms is part of being a responsible open source contributor.
