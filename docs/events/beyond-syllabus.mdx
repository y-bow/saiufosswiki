# Beyond the Syllabus: Open Source in the Real World

**Date:** January 30, 2026  
**Format:** In-person  
**Venue:** SAI University, Chennai  
**Speaker:** Siddharth Shivkumar ([@Suslime](https://forum.fossunited.org/u/suslime))

## Overview

We hosted a session on "Open Source and Learning Beyond the 
Syllabus" conducted by Siddharth Shivkumar. The session was 
highly practical and focused on how open source actually works 
in the real world.

Topics covered:
- Git & GitHub fundamentals
- How pull requests and contributions work
- Using MkDocs for documentation
- How these tools are used in real-world open-source projects

The session was open to students from all departments, and it 
was great to see students asking questions and understanding 
how open source fits beyond classroom learning.

The talk was inspired by the [WikiSyllabus initiative](https://forum.fossunited.org/t/making-college-curricula-accessible/6949) — 
an open-source Wikipedia of university syllabi — which led to 
great conversations about making education more open.

Overall, it was an insightful session and a strong step towards 
building an open-source culture at Sai University.
