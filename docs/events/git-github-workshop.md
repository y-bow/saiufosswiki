# Hands-on Git & GitHub Workshop for Beginners

**Date:** September 17, 2025
**Format:** In-person Workshop
**Organized by:** FOSS Club, SAI University

## Overview

An engaging hands-on workshop on Git and GitHub, bringing together students eager to strengthen their foundations in version control and collaborative development.

## What We Covered

- Core Git concepts and practical workflows
- Understanding repositories and commits
- How teams contribute to shared codebases via GitHub
- Why version control is indispensable in modern software development
- Collaboration, transparency, and shared ownership in open source

## Key Takeaway

Students actively experimented, asked thoughtful questions, and gained confidence using tools that power both open-source and industry projects.
